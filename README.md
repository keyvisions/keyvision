PVK emulator (Programmable Visual Keyboard)
===

   - Raspberry PI Zero W
   - 1920x540 Display (0.5K) 113dpi
   - USB powered
   - Keyboard layouts
   - node.js + sockets: listen for layout, font, key set
   - Bluetooth*
   - WiFi*
   - Camera*
   - Speakers*
   - Microphone*

Keyboard layout
===

Key type (1|2|3) (Special|Regular|Hybrid)
